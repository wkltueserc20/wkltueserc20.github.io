# 👶 育兒助手 - 功能與規格手冊 (v9.4)

本文件詳述了「育兒助手」網頁應用的目前版本功能與 LINE 通知設定注意事項。

## 🛠 v9.4 更新內容 (2026-03-10)
1. **跨日資料連貫性優化**：現在同步時會同時下載「昨今兩日」的備份檔並進行 LWW 合併。解決了午夜跨日後介面短暫空白的問題，確保多人協作的連貫性。
2. **睡眠動態監測橫幅 (Live Banner)**：首頁新增「正在睡覺」狀態監測。若有進行中的睡眠，會顯示已睡時長並提供一鍵「起來了 ☀️」按鈕。
3. **雲端資料夾自動管理**：Google Drive 備份現在會自動收納於「育兒助手備份」資料夾，不再散落於根目錄。

## 🛠 v9.3 更新內容 (2026-03-10)
1. **統計時間區間切換**：在統計分頁新增了「7天 / 14天 / 28天」一鍵切換功能。家長可以根據需求，從短期細節切換到長期趨勢視角。
2. **圖表智慧適配**：當切換至長天數模式時，系統會自動優化 X 軸日期字體與長條圖寬度，確保在手機端依然具備極佳的可讀性。

## 🛠 v9.2 更新內容 (2026-03-10)
1. **現代化 FAB 導覽佈局**：底部導覽列重構為「2 + 1 + 2」結構。正中央新增圓形「＋」動作按鈕 (Floating Action Button)，大幅提升單手操作的便利性。
2. **底部抽屜式表單 (Bottom Sheet)**：所有新增與修改紀錄的操作，現在皆改由底部滑出的抽屜呈現。主畫面空間完整釋放，提供更清爽的瀏覽體驗。
3. **絲滑手勢互動 (Swipe Actions)**：
    * **左滑刪除**：向左滑動紀錄卡片即可揭示刪除按鈕，並配合二次確認防呆。
    * **全域點擊編輯**：直接點擊卡片任何位置即可觸發修改，移除冗餘的按鈕圖示。
4. **iOS 穩定性強化**：實作了「軸向鎖定 (Axis Locking)」，解決 iPhone 上斜向滑動產生的震動不順感；同時鎖死水平溢出，防止誤觸系統側滑返回。
5. **智能數據洞察 (Beta)**：在首頁統計卡片加入昨日數據對比（如：較昨天增加 50ml），並自動分析當日「最長的一覺」與成長紀錄的新鮮度提醒。

## 🛠 v9.1 更新內容 (2026-03-10)
1. **GAS 同步代理架構 (GAS Proxy)**：全面重構同步機制。現在由您的專屬 GAS 腳本持有長效 `refresh_token`，徹底解決了 iOS Safari/PWA 環境下頻繁要求登入 Google 帳號的問題。
2. **長效授權機制**：改用授權碼流 (Authorization Code Flow)。使用者只需在設定中完成一次授權，即可享受數月甚至更長期的無感背景同步。
3. **安全性強化**：引入 `SYNC_SECRET` 通訊密鑰，確保 App 與 GAS 之間的資料代理連線受到保護。
4. **啟動順序優化**：修復了 App 啟動時資料加載順序導致的同步競爭問題，確保啟動即同步的穩定性。

## 🚀 核心紀錄與互動功能
### 1. 互動模式 - **原生 App 級體感**
- **快速新增**：點擊底部中央 **「＋」** 鈕，隨時隨地開啟紀錄抽屜。
- **直覺編輯**：在紀錄清單中 **直接點擊卡片** 即可進行修改。
- **手勢刪除**：對紀錄 **向左滑動** 揭示垃圾桶，操作流暢且高效。

### 2. 餵奶紀錄 (Feeding) - **支援雲端提醒**
- **自動時間**：切換功能時自動代入當下時刻。
- **下餐倒數**：根據最近紀錄自動計算 4 小時倒數。
- **☁️ 雲端排程 (GAS)**：新增或修改紀錄時，同步至 Google Apps Script 雲端。
- **溫馨通知**：通知包含時間段（如：下午 3:30 喝奶時間到了）。

### 3. 數據洞察 (Insights)
- **昨日對比**：統計卡片下方自動顯示與前一日的數值差距。
- **睡眠分析**：自動抓取當日最長的一次連續睡眠（長覺）。
- **發育提醒**：顯示距上次測量身高體重已過天數，適時提醒更新。

## 🛠 技術規格
