# 👶 育兒助手 - 功能與規格手冊 (v9.1)

本文件詳述了「育兒助手」網頁應用的目前版本功能與 LINE 通知設定注意事項。

## 🛠 v9.1 更新內容 (2026-03-10)
1. **GAS 同步代理架構 (GAS Proxy)**：全面重構同步機制。現在由您的專屬 GAS 腳本持有長效 `refresh_token`，徹底解決了 iOS Safari/PWA 環境下頻繁要求登入 Google 帳號的問題。
2. **長效授權機制**：改用授權碼流 (Authorization Code Flow)。使用者只需在設定中完成一次授權，即可享受數月甚至更長期的無感背景同步。
3. **安全性強化**：引入 `SYNC_SECRET` 通訊密鑰，確保 App 與 GAS 之間的資料代理連線受到保護。
4. **啟動順序優化**：修復了 App 啟動時資料加載順序導致的同步競爭問題，確保啟動即同步的穩定性。

## 🛠 v9.0 更新內容 (2026-03-09)
1. **結構模組化重構**：將單一大型檔案拆分為 `components`, `hooks`, `utils` 與 `types` 定義，大幅提升代碼可讀性、擴充性與型別安全性。
2. **智慧雙向同步與多人協作**：全面升級 Google Drive 同步機制。現在支援多設備「拉取 -> 合併 -> 推送」流程，不同手機紀錄的資料會自動彙整而不互相覆蓋，實現真正的多人共同紀錄。
3. **PWA 完整離線支援**：引入 Service Worker 技術，支援離線開啟 App。即便在完全斷網環境（如地下室），也能秒開介面並進行紀錄，連線後自動併入雲端。
4. **事件驅動自動同步**：當您回到 App 視窗、切換分頁或網路恢復時，系統會安靜地在背景執行靜默同步，右上角新增同步狀態指示圖標（Syncing / Updated）。
5. **移除尿布功能**：為了精簡流程，移除了尿布紀錄功能，並將首頁摘要重心移至奶量與最新成長數據（方案 B 佈局）。

## 🛠 v8.9 更新內容 (2026-03-02)
1. **每日獨立備份檔**：備份機制升級。現在每天會產生一個獨立的 CSV 檔案（例如：`baby_records_20260302.csv`），方便您按日期追蹤歷史紀錄，不再只是單一檔案覆蓋。
2. **自動同步邏輯**：系統會自動辨識當天是否已建立備份，若已存在則更新內容，若不存在則建立新檔。

## 🚀 核心紀錄功能
### 1. 餵奶紀錄 (Feeding) - **支援雲端提醒**
- **自動時間**：切換功能時自動代入當下時刻。
- **下餐倒數**：根據最近紀錄自動計算 4 小時倒數。
- **☁️ 雲端排程 (GAS)**：新增或修改紀錄時，同步至 Google Apps Script 雲端。
- **溫馨通知**：通知包含時間段（如：下午 3:30 喝奶時間到了）。

### 2. 睡眠紀錄 (Sleep) - **修復完整**
- **歸屬日期**：以「起床時間」決定紀錄歸屬哪一天（跨夜覺計算）。
- **起訖計時**：點擊「開始睡覺」記錄起點，點擊「起來了 ☀️」自動計算總時數並生成備註。

### 3. 成長紀錄 (Growth)
- **成長**：紀錄身高 (cm) 與體重 (kg)，支援雙軸成長曲線圖。
- **最新摘要**：首頁直接顯示最新的體重與身高數值。

## 📢 LINE 通知設定與多人接收指南
- LINE 的 `Push Message` 是傳送給特定的 **User ID** (U 開頭的編碼)。
- **解決方案**：每個人的 User ID 都不同。若要各自接收，每個人手機上的 App 設定頁都必須填入各自的 User ID。

### 2. 群組通知方案 (最推薦)
- **做法**：建立一個 LINE 群組，將爸爸、媽媽與 **LINE 機器人** 全部拉進去。
- **優點**：機器人只需發送 **1 則** 訊息，群組內所有人都看得到。
- **設定**：在設定頁的 User ID 欄位填入該群組的 **Group ID** (C 開頭)。

### 3. 廣播模式 (Broadcast)
- **做法**：修改 GAS 程式碼，將 API 路徑改為 `/message/broadcast`。
- **效果**：只要有加入機器人好友的人，通通都會收到訊息。
- **代價**：訊息扣除額度 = 好友數 * 餵奶次數。

### ⚠️ 免費額度警示
- LINE 官方帳號免費方案每月僅有 **200 則** 訊息額度。
- 若使用「個人推播」且人數眾多，額度會快速耗盡導致通知失效。強烈建議使用 **「群組通知」** 以節省額度。
