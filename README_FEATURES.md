## 🚀 核心紀錄功能
### 1. 餵奶紀錄 (Feeding) - **支援雲端提醒**
- **自動時間**：切換功能時自動代入當下時刻。
- **下餐倒數**：根據最近紀錄自動計算 4 小時倒數。
- **☁️ 雲端排程 (GAS)**：新增或修改紀錄時，同步至 Google Apps Script 雲端。
- **溫馨通知**：通知包含時間段（如：下午 3:30 喝奶時間到了）。

### 2. 睡眠紀錄 (Sleep) - **修復完整**
- **歸屬日期**：以「起床時間」決定紀錄歸屬哪一天（跨夜覺計算）。
- **起訖計時**：點擊「開始睡覺」記錄起點，點擊「起來了 ☀️」自動計算總時數並生成備註。

### 3. 尿布與成長紀錄
- **尿布**：紀錄尿尿、便便、都有、乾爽四種狀態。
- **成長**：紀錄身高 (cm) 與體重 (kg)，支援雙軸成長曲線圖。

## 📢 LINE 通知設定與多人接收指南
要讓多人同時收到通知，請參考以下注意事項：

### 1. User ID 的專屬性
- LINE 的 `Push Message` 是傳送給特定的 **User ID** (U 開頭的編碼)。
- **解決方案**：每個人的 User ID 都不同。若要各自接收，每個人手機上的 App 設定頁都必須填入各自的 User ID。

### 2. 群組通知方案 (最推薦)
- **做法**：建立一個 LINE 群組，將爸爸、媽媽與 **LINE 機器人** 全部拉進去。
- **優點**：機器人只需發送 **1 則** 訊息，群組內所有人都看得到。
- **設定**：在設定頁的 User ID 欄位填入該群組的 **Group ID** (C 開頭)。

### 3. 廣播模式 (Broadcast)
- **做法**：修改 GAS 程式碼，將 API 路徑改為 `/message/broadcast`。
- **效果**：只要有加入機器人好友的人，通通都會收到訊息。
- **代價**：訊息扣除額度 = 好友數 * 餵奶次數。

### ⚠️ 免費額度警示
- LINE 官方帳號免費方案每月僅有 **200 則** 訊息額度。
- 若使用「個人推播」且人數眾多，額度會快速耗盡導致通知失效。強烈建議使用 **「群組通知」** 以節省額度。
